**Narration (Icelandic):**

- Show orders list with columns:
  - Pöntunarnúmer (Order #): P-2024-001
  - Viðskiptavinur (Customer): Kaffi Hús ehf.
  - Gámur (Container): C-1189
  - Áætlaður dagur (Scheduled): 2024-01-15
  - Staða (Status): Áætluð (Scheduled)

## SECTION 2: CREATE NEW ORDER (0:25 - 0:55)

**Narration (Icelandic):**
"Til að búa til nýja pöntun skaltu smella á 'Ný pöntun'. Fyrst velurðu viðskiptavin úr listanum."

**Narration (English subtitle):**
"To create a new order, click 'New Order'. First, select a customer from the list."

**Screen:**
- Click "Ný pöntun" button
- Show form step 1:
  - Viðskiptavinur: Dropdown with customers
  - Select "Kaffi Hús ehf."
  - Show customer's containers below

**Narration (Icelandic):**
"Síðan velurðu hvaða gám á að tæma. Þú sérð fyllingarhlutfall hvers gáms til að forgangsraða."

**Narration (English subtitle):**
"Then select which container to empty. You see the fill level of each container to prioritize."

**Screen:**
- Show container selection:
  - C-1189 (Laugavegur 12) - 85% fullur ⚠️ High priority
  - C-1190 (Laugavegur 14) - 45% fullur
  - C-1191 (Laugavegur 16) - 30% fullur
- Select C-1189

**Caption on screen:**
"Veldu gám → Skoða fyllingarhlutfall → Forgangsraða"
"Select container → View fill level → Prioritize"

**Narration (Icelandic):**
"Veldu dagsetningu fyrir tínsluna og bættu við athugasemd ef þörf er á."

**Narration (English subtitle):**
"Select a date for the collection and add a note if needed."

**Screen:**
- Date picker: Select "15. janúar 2024"
- Optional note field: "Gámur mjög fullur, forgangsmál"
- Click "Vista pöntun" (Save Order)

## SECTION 3: ASSIGN TO DRIVER AND ROUTE (0:55 - 1:20)

**Narration (Icelandic):**
"Eftir að pöntunin er búin til þarftu að úthluta henni til ökumanns og leiðar. Smelltu á pöntunina til að opna upplýsingar."

**Narration (English subtitle):**
"After creating the order, you need to assign it to a driver and route. Click on the order to open details."

**Screen:**
- Click on newly created order P-2024-001
- Show order detail modal

**Narration (Icelandic):**
"Veldu leið úr listanum. Kerfið leggur til bestu leiðina út frá staðsetningu gámsins."

**Narration (English subtitle):**
"Select a route from the list. The system suggests the best route based on container location."

**Screen:**
- Route dropdown with suggestions:
  - Leið 1: Miðborg (Recommended) ✓
  - Leið 2: Vesturbær
  - Leið 3: Austurbær
- Select "Leið 1: Miðborg"

**Narration (Icelandic):**
"Síðan velurðu ökumann sem er í vinnu þann daginn."

**Narration (English subtitle):**
"Then select a driver who is working that day."

**Screen:**
- Driver dropdown:
  - Jón Jónsson (Available) ✓
  - Guðrún Pétursdóttir (Off duty)
- Select "Jón Jónsson"
- Click "Vista úthlutun" (Save Assignment)

**Caption on screen:**
"Pöntun → Leið → Ökumaður → Vista"
"Order → Route → Driver → Save"

## SECTION 4: TRACK ORDER STATUS (1:20 - 1:40)

**Narration (Icelandic):**
"Þú getur fylgst með stöðu pöntunarinnar í rauntíma. Þegar ökumaður byrjar að vinna pöntunina breytist staðan í 'Í vinnslu'."

**Narration (English subtitle):**
"You can track the order status in real-time. When the driver starts working on the order, the status changes to 'In Progress'."

  - 11:15 - Lokið (Completed) → Green

"Þegar pöntunin er lokið sérðu myndir sem ökumaðurinn tók og athugasemdir."

**Narration (English subtitle):**
"When the order is completed, you see photos the driver took and their notes."

**Screen:**
- Show completed order details:
  - Photos: Before (full container), After (empty container)
  - Driver notes: "Gámur tæmdur án vandamála"
  - Completion time: 11:15

**Caption on screen:**
"Lokið → Myndir → Athugasemdir → Tími"
"Completed → Photos → Notes → Time"
